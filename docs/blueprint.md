# **App Name**: MotionFlow

## Core Features:

- Motion Photo Upload: Allow users to upload Motion Photo files (e.g., .jpg).
- Data Stream Processing: Process the Motion Photo data stream in the browser.
- Static Image Extraction: Extract the static .jpg image from the Motion Photo.
- Short Video Extraction: Extract the short .mp4 video from the Motion Photo.
- In-Browser Download: Allow users to directly download the extracted .jpg and .mp4 files in the browser.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to suggest technology and clarity.
- Background color: Light gray (#F0F4F8), close to white, for a clean interface.
- Accent color: A vivid orange (#FFAB40) to draw attention to key controls.
- Body and headline font: 'Inter', a sans-serif font that offers a modern, neutral feel.
- Simple, intuitive layout for easy file uploading and processing.
- Clear and recognizable icons for actions like upload, extract, and download.
- Subtle loading animations to indicate processing progress.
